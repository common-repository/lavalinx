== Installation ==

1. Upload the `lavalinx` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= What is a sending page? =

Sending pages are pages that point links to other websites

= What is a receiving page? =

Receiving Pages are pages that links are pointed to. They are usually homepages or other primary pages in your site.
